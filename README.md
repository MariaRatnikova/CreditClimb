# CreditClimb

CreditClimb is a web application designed to help university students track their academic progress in a clear and structured way.  
It provides transparency about completed modules, remaining credits, and overall study advancement, while also supporting BAföG-related documentation processes.

---

## Overview

Many students lack a centralized and structured overview of their academic status.  
CreditClimb organizes relevant study information and presents it in a user-friendly interface that supports both self-management and communication with academic advisors.

---

## Problem Statement

Students need:

- A clear overview of completed and pending modules  
- Transparency regarding accumulated and missing credits  
- Structured semester planning  
- Support when preparing BAföG Leistungsnachweis  

Without a centralized system, this information is often fragmented and difficult to manage.

---

## Key Features

- Semester-based module overview  
- Credit progress visualization  
- Detailed module information pages  
- Structured academic tracking  
- Support for BAföG documentation  
- Transparent overview for advisors and students
